# Hi, I'm Athan S Maliakkal

Physics undergraduate and systems programmer focused on low-level software and operating system architecture. I work close to the hardware—building kernels, experimenting with multicore systems, memory management, device drivers, and secure execution models. My work involves Rust, C++, x86_64 assembly, and bare-metal development across UEFI environments, virtualization platforms, and custom tooling. I approach systems engineering with a physics mindset: understanding the fundamental constraints of machines and designing clean, reliable abstractions on top of them.
